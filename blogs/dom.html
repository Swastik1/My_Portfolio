  • What is DOM ?

    The HTML DOM is a standard object model and object-oriented HTML document representation in a logical tree structure where each node contains an object representing a part of a document. It contains: HTML elements as objects Properties of all HTML elements Methods to access all HTML elements Events for all HTML elements The HTML DOM is a standard for how to get, change, add, or delete HTML elements.     How to use the document object to access and manipulate HTML.


    document.getElementByID(id) - Find an element by element id
    document.getElementByTagName(name) - Find elements by tag name
    document.getElementByClassName(name)- Find elements by class name

  • Adding and Deleting Elements

    document.createElement(element)Create an HTML element
    document.removeChild(element)Remove an HTML element
    document.appendChild(element)Add an HTML element
    document.replaceChild(new, old)Replace an HTML element
    document.write(text)Write into the HTML output stream
